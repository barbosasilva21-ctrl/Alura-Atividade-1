# 🌱 Projeto Agrinho: Sustentabilidade e Inovação no Campo

## 📖 Descrição
Este projeto é um site desenvolvido para a iniciativa do Projeto Agrinho. O foco principal da página é conscientizar a comunidade sobre a importância da agricultura sustentável e a preservação do meio ambiente, garantindo um futuro mais verde.

## 🎯 Objetivo
O objetivo deste projeto é:
* Informar o público sobre práticas sustentáveis no campo.
* Aplicar os conhecimentos adquiridos em desenvolvimento web (Front-end).
* Entregar uma interface amigável, responsiva e de fácil navegação para a avaliação.

## 🛠️ Recursos Técnicos Utilizados
A interface foi construída utilizando as seguintes linguagens base da web:
* **HTML5:** Para a estruturação semântica do conteúdo.
* **CSS3:** Para a estilização, layout em cartões e responsividade da página.
* **JavaScript:** Para interatividade (eventos de clique e alertas).
